# 651372.xyz — 周易起卦

离线可用的金钱卦起卦网页应用。用户模拟三枚硬币投掷六次，生成六爻，显示本卦、变卦、动爻及对应的卦辞爻辞。

## Language

**金钱卦 / 起卦**：
六次投掷三枚硬币的占卜方法。每次三枚硬币的和（字面=3，花面=2）决定该爻：6=老阴（变爻）、7=少阳（不变）、8=少阴（不变）、9=老阳（变爻）。
_Avoid_：卜卦、占卜（过于宽泛）

**本卦**：
由六次投掷结果直接构成的六十四卦之一。六爻从下往上排列（初爻→上爻）。

**变卦**：
将本卦中的动爻（老阴/老阳）翻转（阴变阳、阳变阴）后得到的新卦。

**动爻**：
本卦中值为6（老阴）或9（老阳）的爻，表示该位有变化动态。默认只显示动爻对应的爻辞，可通过 toggle 展开全部爻辞。

**卦辞**：
一卦整体的断语。如「乾：元亨，利贞」。

**爻辞**：
每一条爻的断语。如「初九：潜龙，勿用」。

**爻性**：
阳爻用⚊表示（对应九），阴爻用⚋表示（对应六）。初爻至五爻按位置称呼（初九、九二、九三…上九；初六、六二…上六）。

**摇卦**：
用户点击按钮依次获得六次结果的过程。每次点击触发 0.8 秒数字闪烁动画后定格，按钮在动画期间禁用。
